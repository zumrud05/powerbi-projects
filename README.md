# Personal Finance Analysis – Power BI

This project analyzes personal income and expenses based on bank card statements.

## Objective
- Track monthly income and expenses
- Categorize spending behavior
- Identify major expense categories
- Analyze financial trends over time

## Data Description
- Source: Bank card statements (anonymized)
- Time period: Monthly data
- Categories: Food, Transport, Utilities, Shopping, Entertainment, Others


## Tools Used
- Power BI
- Sql
- Microsoft Excel

## Key Insights
- Monthly expense distribution by category
- Income vs expense comparison
- Top spending categories
- Trend analysis over months

## Files
- `.pbix` – Power BI dashboard
- `.xlsx` – Cleaned dataset
